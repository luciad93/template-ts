User Guide

Button Functions:

  • Light: Toggles the backlight of the watch display.
  • Mode: Cycles through the modes: View Time, Set Hours, Set Minutes.
  • Increase: In Set Hours mode, increases the hour. In Set Minutes mode, increases the minute.
  • Reset: Resets the current setting operation.
  • Format: Toggles between 12-hour and 24-hour time formats.

Timeouts: After a period of inactivity, the watch will automatically return to View Time mode to save power and simplify usage.
